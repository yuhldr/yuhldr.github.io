жіўеҠЁж–№зЁӢжұӮи§Ј

еҸ‘иЎЁдәҺ 2021-03-15 жӣҙж–°дәҺ 2022-06-28 еҲҶзұ»дәҺ зү©зҗҶ йҳ…иҜ»ж¬Ўж•°пјҡ жң¬ж–Үеӯ—ж•°пјҡ 945 йҳ…иҜ»ж—¶й•ҝ ≈ 3 еҲҶй’ҹ
еӨҚд№ пјҲйў„д№ пјүеӨ§еӯҰжң¬з§‘ж•°еӯҰзү©зҗҶж–№жі•пјҢжіўеҠЁж–№зЁӢзҡ„жұӮи§Ј

еӨ§йҮҸеҸӮиҖғгҖҠж•°еӯҰзү©зҗҶж–№жі•гҖӢ-жқЁеӯ”еәҶ

ејҰжҢҜеҠЁжҺЁеҜјжіўеҠЁж–№зЁӢ

й«ҳдёӯз«һиөӣзү©зҗҶеёёи§Ғиҝ‘дјјпјҢxи¶Ӣиҝ‘дәҺ0ж—¶пјҢеҰӮдёӢпјҲз”»дёӘдёүи§’еҪўжҳҫиҖҢжҳ“и§Ғпјүпјҡ

$$cos x = 1 $$

$$ sin x = tan x \times cos x = tanx = x(еј§еәҰеҲ¶)$$

еҲқе§Ӣи®ҫе®ҡ

ејҰпјҡзәҝеҜҶеәҰ $\rho$пјҢжҜҸдёҖзӮ№дёҠдёӢйңҮеҠЁпјҲxж–№еҗ‘е”ҜдёҖдёә0пјү

еҸ—еҠӣеҲҶжһҗ

  1. xеӨ„пјҢdxй•ҝзҡ„зҡ„дёҖж®өз»іеӯҗпјҲзәҝеҜҶеәҰ$\rho$пјү:

    еҸ—еҲ°дёүдёӘеҸҜиғҪзҡ„еҠӣпјҢдёӨз«Ҝзҡ„жӢүеҠӣ$T_x$е’Ң$T_{x+dx}$пјҢдёҺз»іеӯҗзӣёеҲҮпјҢеҲҮи§’еӨ§е°ҸеҲҶеҲ«дёә$\alpha$е’Ң$\beta$пјҢе№¶дё”еҚ•дҪҚй•ҝеәҰејҰеҸҜиғҪеҸ—еҲ°еӨ–еҠӣпјҢжҡӮж—¶и®ӨдёәжҳҜз«–зӣҙж–№еҗ‘пјҢ$f_0(x,t)$

  2. з”ұдәҺxиҪҙдёҠжІЎжңүиҝҗеҠЁпјҢxж–№еҗ‘еҸ—еҠӣе№іиЎЎ

    $$T_xcos\alpha - T_{x+dx} cos\beta = 0$$

    ејҰзҡ„жҢҜе№…еҫҲе°ҸпјҢеҸҜеҸ–$\alpha$$\beta$ -> 0пјҢз”ұдёҠйқўзҡ„иҝ‘дјјеҸҜеҫ—

    $$T_x = T_{x+dx} = T$$

  3. з«–зӣҙж–№еҗ‘дёҠпјҢиҝҗеҠЁж–№зЁӢдёә

    $$Tsin\alpha-T_{x+dx}sin\beta+f_0(x,t)dx=ma=(\rho dx) u_{tt}$$

    е…¶дёӯ$u_{tt}$дёәз«–зӣҙж–№еҗ‘дҪҚз§»еҜ№ж—¶й—ҙзҡ„дәҢйҳ¶еҜјж•°

    з”ұжңҖејҖе§Ӣзҡ„иҝ‘дјје…ізі»пјҢеҸҜиҝӣдёҖжӯҘеҢ–з®Җдёәпјҡ

    $$ Ttan\alpha-Ttan\beta+f_0(x,t)dx=(\rho dx) u_{tt} $$

    $tan$еҮҪж•°еҸҜд»ҘзңӢеҒҡж–ңзҺҮпјҡ

    $$Ttan\alpha-Ttan\beta=T(\frac{\partial u}{\partial x}\bigg|_{x+dx}-\frac{\partial u}{\partial x}\bigg|_x)$$

    е…¶дёӯ

    $$\frac{\partial u}{\partial x}\bigg|_{x+dx} - \frac{\partial u}{\partial x}\bigg|_x=\frac{\partial^2 u}{\partial^2 x}dx$$
    еҸҜеҫ—

    $$T\frac{\partial^2 u}{\partial^2 x}+f_0(x,t)=\rho u_{tt}$$

    еҶҷжҲҗж ҮеҮҶеҪўејҸпјҡ

    $$u_{tt}-au_{xx}=f(x,t)$$

    е…¶дёӯ $a=\frac{T}{\rho}$пјҢеҚіжҢҜеҠЁеңЁејҰдёҠдј ж’ӯзҡ„йҖҹеәҰпјҢ$f(x,t)=\frac{f_0(x,t)}{\rho}$иў«з§°дёәеӨ–еҠӣеҜҶеәҰпјҢеҚ•дҪҚй•ҝеәҰеҜҶеәҰдёҠжүҖж–ҪеҠ зҡ„еӨ–еҠӣгҖӮеҪ“$f(x,t)=0$пјҢж–№зЁӢеҸҳдёә

    $$u_{tt}-a^2u_{xx}=0 (0<x<l,t >0)$$

    еҚідёәдёҖж¬ЎйҪҗж¬ЎжіўеҠЁж–№зЁӢ

е®ҡи§ЈжқЎд»¶

вҖңеҲқе§ӢжқЎд»¶вҖқдёҺвҖңиҫ№з•ҢжқЎд»¶вҖқ

еҲқе§ӢжқЎд»¶

дҪҚз§»еҲҶеёғе’ҢйҖҹеәҰеҲҶеёғе·ІзҹҘпјҢ$\psi(x)$е’Ң$v(x)$дёәе·ІзҹҘеҮҪж•°

$$u(x,0)=\psi(x) \qquad 0 \leq x \leq l$$

$$u_t(x,0)=v(x) \qquad 0 \leq x \leq l$$

иҫ№з•ҢжқЎд»¶

е…ұдёүзұ»пјҢеҲҶеҲ«жҳҜпјҡ

  • з»іеӯҗдёӨз«Ҝеӣәе®ҡ
  • з»іеӯҗдёӨз«ҜдёҚж”¶еҲ°д»»дҪ•еҠӣпјҢиҮӘз”ұз«Ҝ
  • з»іеӯҗдёӨз«ҜжҳҜеј№з°§

  1. еӣәе®ҡиҫ№з•ҢпјҲдёӨз«Ҝпјү

    еҘҪжҜ”жңҖдёҠйқўзҡ„йӮЈдёӘеӣҫпјҢдёӨз«Ҝе§Ӣз»ҲеңЁxиҪҙдёҠзҡ„жҹҗдёҖзӮ№еҚі

    $$u(0,t)=u(l,t)=0$$

    йҡҸзқҖж—¶й—ҙtзҡ„еҸҳеҢ–пјҢx=0е’Ңx=lеӨ„зҡ„з»іеӯҗдёӨз«Ҝеқҗж ҮuдёҚеҸҳ

  2. иҮӘз”ұиҫ№з•ҢпјҲдёӨз«Ҝпјү

    еңЁжҲ‘д»¬д№ӢеүҚзҡ„ж–№зЁӢжҺЁеҜјдёӯпјҢй»ҳи®Өзҡ„жҳҜпјҢ$x$еӨ„зҡ„з»іеӯҗй•ҝеәҰ$dx$пјҢеҸ—еҲ°з»іеӯҗдёӨз«Ҝзҡ„е…¶д»–з»іеӯҗжӢүеҠӣдёә

    $$T_xгҖҒT_{x+dx}$$

    $y$иҪҙж–№еҗ‘зҡ„еҸ—еҠӣпјҢxиҪҙж–№еҗ‘дёҖеҫӢи®ӨдёәдёҚеҠЁ
    $$T_xsin\alphaгҖҒT_{x+dx}\beta$$

    иҝ‘дјјд»ҘеҗҺдёә

    $$T_xsin\alpha = Ttan\alpha = T\frac{\partial u}{\partial x}\bigg|_x$$

    $$T_{x+dx}sin\beta = Ttan\beta = T\frac{\partial u}{\partial x}\bigg|_{x+dx}$$

    иҖҢзңҹе®һжғ…еҶөжҳҜпјҢ$x=0$е’Ң$x=l$еӨ„пјҢз»іеӯҗиҮӘз”ұж—¶пјҢиҝҷдёӨж®өз»іеӯҗеҸӘиғҪеҸ—еҲ°дёҖз«Ҝз»іеӯҗзҡ„еҠӣпјҢеҸҰдёҖз«ҜеӨ–еҠӣдёә0пјҢеҚі

    $$T\frac{\partial u}{\partial x}\bigg|_0=0$$

    $$T\frac{\partial u}{\partial x}\bigg|_l=0$$

    е…¶дёӯTеҸҜд»ҘеҺ»жҺүпјҲз»іеӯҗдёӨз«ҜдёҖдёӘеӨ–еҠӣе·ҰдёӢпјҢдёҖдёӘеҸідёҠпјҢж–№еҗ‘дёҚеҗҢпјҢдҪҶжҳҜжӯӨеӨ„дёә0пјҢдёҚиҖғиҷ‘жӯЈиҙҹеҸ·пјү

    $$\frac{\partial u}{\partial x}\bigg|_0=0$$

    $$\frac{\partial u}{\partial x}\bigg|_l=0$$

  3. еј№жҖ§иҫ№з•ҢпјҲдёӨз«Ҝпјү

    з»іеӯҗеңЁдёӨдёӘз«–зӣҙзҡ„еј№з°§дёҠзі»зқҖпјҢеңЁиҝҷз§Қжғ…еҶөдёӢпјҢ2дёӯзҡ„з«–зӣҙж–№еҗ‘еӨ–еҠӣдёҚдёә0пјҢи®ӨдёәжҳҜдёҖз§Қеј№еҠӣпјҢзұ»дјјдәҺеј№з°§$f=au$пјҢе…¶дёӯaдёәеј№з°§зі»ж•°пјҢuдёәyиҪҙз«–зӣҙж–№еҗ‘дҪҚз§»пјҢе№іиЎЎж—¶пјҢдёӨз«ҜеҸ—еҠӣдёәпјҲжҡӮж—¶и®Өдёәеҗ‘дёҠзҡ„еҠӣдёәиҙҹпјүпјҡ

    $$-\frac{\partial u}{\partial x}\bigg|_0 + a_1u(0,t) = 0$$

    $$\frac{\partial u}{\partial x}\bigg|_l + a_2u(l,t) =0$$

    дёӨиҫ№еј№з°§еј№жҖ§зі»ж•°дёҚдёҖж ·пјҢеҲҶеҲ«дёә$a_1$пјҢ$a_2$гҖӮ
